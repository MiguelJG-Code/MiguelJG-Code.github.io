1.Project Name: Foursight Car Rental

   We were given a task to find a way to implement Barcode reader, Qr generator, and sents email. So we decided to maake a car rental.

2. What it does

   A console-based Car Rental Management System written in C that uses text files as its database, supports Admin and Cashier accounts, generates barcodes and receipts, and can send email confirmations. 

3. Built with
   This code implements a full-featured console Car Rental Management System with:

* Admin/Cashier authentication
* Car inventory CRUD
* Customer CRUD
* Bulk car rentals
* Return processing
* Payment and billing
* Receipt generation
* Barcode generation (ASCII + PNG)
* Email notifications
* Reports
* User management
* Password management
* Automatic data persistence using text databases
* Cross-platform console support (Windows/Linux)

It is essentially a complete offline Car Rental Management System built in C with integrated inventory, customer, rental, billing, reporting, barcode, and email capabilities.
