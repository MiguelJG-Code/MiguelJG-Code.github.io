1. PigMart — Online Pig Marketplace

A full-stack web marketplace for buying pigs online, built for Samar region sellers.
Live site: [Visit PigMart]([https://your-live-site-url.com])

2. What it does
Customers can browse available pigs, filter by breed and size, and submit 
purchase requests with delivery details. Admins manage listings, confirm 
or decline orders, and update store settings through a secure admin panel.

3. Built with
- HTML / CSS / JavaScript (Frontend)
- PHP (Backend API)
- MySQL (Database)
- InfinityFree (Hosting)

4. Features
- Live pig listings with photos, breed, weight, and price
- Customer request form with delivery and payment info
- Admin login with token-based authentication
- Admin dashboard to confirm, decline, and manage orders
- Image upload support for pig listings

5. What I learned
- How to build a REST API in PHP
- Connecting a frontend to a live database
- Admin authentication and session handling
- Deploying a full-stack app on shared hosting
